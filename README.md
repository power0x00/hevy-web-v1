# Hevy Web

A beautiful, Apple-like workout tracking app for the web, inspired by the Hevy mobile app.

## âœ¨ Features

- **Workout Logging**: Track exercises, sets, reps, and weight with an intuitive interface
- **Exercise Library**: 300+ built-in exercises organized by muscle group
- **Routines**: Create and save reusable workout templates
- **Progress Tracking**: View workout history and track your progress over time
- **Rest Timer**: Built-in rest timer between sets
- **Dark Mode**: Beautiful dark mode support
- **Offline-First**: All data stored locally in your browser
- **Mobile-Friendly**: Responsive design that works on all devices

## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/hevy-web-v1.git

# Navigate to the project directory
cd hevy-web-v1

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.
